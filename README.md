# Circuit Annotator

一个用于电路图标注的图形化工具，支持矩形/多边形组件标注、端口标注、连接关系标注，以及层级结构管理。

## 功能特性

### 1. 组件标注
- **矩形组件**：使用快捷键 `W` 或工具栏按钮，拖拽绘制矩形框
- **多边形组件**：使用快捷键 `E` 或工具栏按钮，点击创建多边形顶点，右键完成绘制
- **自动命名**：组件自动命名为 `类型_X坐标_Y坐标` 格式，避免重名
- **层级颜色**：不同层级的组件使用不同颜色区分（Level 0: 蓝色, Level 1: 绿色, Level 2: 橙色, Level 3: 紫色）

### 2. 端口标注
- **添加端口**：使用快捷键 `P` 或工具栏按钮，在组件上点击添加端口
- **端口拖动**：端口只能在所属组件内部拖动，不会超出边界
- **端口吸附**：连接时自动吸附到附近的端口，提高操作精度

### 3. 连接标注
- **创建连接**：使用快捷键 `C` 或工具栏按钮，依次点击两个端口创建连接
- **质心显示**：多个端口连接时显示质心点，表示连接的中心位置
- **颜色编码**：
  - 同层级连接：金色
  - 跨层级连接：根据层级对显示不同颜色（粉、青、绿、红、紫等）
- **连接删除**：
  - 选中质心点删除整个连接组
  - 选中线段删除单个连接（3+端口时只删除该端口与质心的连接）

### 4. 层级管理
- **自动层级检测**：加载时自动检测组件包含关系，建立父子层级
- **层级显示**：右侧面板显示所有层级，点击可筛选显示特定层级
- **父子关系**：
  - 右侧面板显示组件的父组件和子组件
  - 支持多父组件（引用关系）
- **容器展开/收起**：点击容器组件可展开/收起其子组件

### 5. 视图控制
- **缩放**：鼠标滚轮缩放，支持 10% - 500% 缩放范围
- **平移**：按住空格键拖拽平移视图
- **组件选择**：
  - 单击选中组件
  - 选中后其他组件淡化显示
  - 点击空白处取消选择
- **层级筛选**：右侧面板点击层级，只显示该层级组件

### 6. 快捷键
| 快捷键 | 功能 |
|--------|------|
| `W` | 绘制矩形组件 |
| `E` | 绘制多边形组件 |
| `P` | 添加端口 |
| `C` | 创建连接 |
| `R` | 添加引用关系 |
| `Delete` | 删除选中项 |
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `空格+拖拽` | 平移视图 |
| `滚轮` | 缩放视图 |

## 使用方法

### 1. 启动程序
```bash
python main.py
```

### 2. 加载数据
- 点击 **Load Image Folder** 选择包含电路图的文件夹
- 点击 **Load JSON Folder** 选择包含标注文件的文件夹
- 使用左右箭头切换图片

### 3. 创建标注
1. **创建组件**：
   - 按 `W` 进入矩形模式，拖拽绘制
   - 按 `E` 进入多边形模式，点击创建顶点，右键完成

2. **添加端口**：
   - 按 `P` 进入端口模式
   - 在组件上点击添加端口

3. **创建连接**：
   - 按 `C` 进入连接模式
   - 点击第一个端口（变绿色）
   - 点击第二个端口完成连接

4. **添加引用**：
   - 按 `R` 进入引用模式
   - 点击源组件
   - 点击目标组件建立引用关系

### 4. 编辑标注
- **拖动组件**：直接拖拽组件
- **拖动端口**：在组件内部拖拽端口
- **删除**：选中后按 `Delete`
- **修改属性**：右侧面板编辑组件属性

### 5. 保存结果
- 按 `Ctrl+S` 或点击 **Save** 按钮保存当前标注
- 标注自动保存为 JSON 格式

## 数据格式

### JSON 结构
```json
{
  "components": {
    "组件ID": {
      "id": "组件ID",
      "type": "组件类型",
      "shape": {
        "type": "rect|polygon",
        "box": [x1, y1, x2, y2],
        "points": [[x1, y1], [x2, y2], ...]
      },
      "parent": "父组件ID",
      "children": ["子组件ID1", "子组件ID2"],
      "references": ["引用组件ID"],
      "ports": {
        "端口ID": {
          "id": "端口ID",
          "coord": [x, y]
        }
      }
    }
  },
  "external_ports": {
    "端口ID": {
      "id": "端口ID",
      "coord": [x, y]
    }
  },
  "connections": [
    {
      "nodes": [
        {"component": "组件ID", "port": "端口ID"},
        {"component": "external", "port": "端口ID"}
      ]
    }
  ]
}
```

## 注意事项

1. **自动保存**：切换图片前请保存当前标注
2. **层级关系**：大框包含小框时会自动建立父子关系
3. **端口限制**：端口不能拖出组件边界
4. **连接限制**：连接必须发生在两个不同端口之间
